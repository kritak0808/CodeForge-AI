# User Guide

This guide walks you through using CodeForge AI to manage your software development life cycle autonomously.

---

## 1. Authentication & Login

1.  Open the web console at `http://localhost:3000`.
2.  Navigate to the `/login` page.
3.  Enter your username and password. The system will issue a JWT access token and store your session credentials.
4.  Once authenticated, you will be redirected to the Dashboard (Control Center).

---

## 2. Managing Projects

Projects represent the software systems you want CodeForge AI to build or maintain.

1.  Navigate to the **Projects** tab.
2.  Click **Create New Project**.
3.  Fill in the project name, description, budget limit (USD), and tech stack parameters (e.g. FastAPI, Next.js).
4.  Click **Submit** to register your project.

---

## 3. Launching Workflows

Workflows represent execution pipelines containing a sequence of autonomous agent tasks.

1.  Navigate to the **Workflows** page.
2.  Click **Create New Workflow** and select your Project.
3.  Input requirements (e.g. "Build an e-commerce backend with Stripe integration").
4.  Once started, the workflow progresses through autonomous stages:
    *   **Research Agent**: Scans dependency registries.
    *   **Architect Agent**: Formulates architecture design.
    *   **Database Agent**: Generates schemas and migrations.
    *   **Backend Agent**: Writes FastAPI endpoints.
    *   **Frontend Agent**: Scaffold UI pages.
    *   **QA Agent**: Generates test specs.
    *   **Security Agent**: Scans code for vulnerability threats.
    *   **DevOps Agent**: Packages Docker containers.

---

## 4. Manual Approval Gates

For safety and quality compliance, certain stages (such as Database migrations or Deployment actions) halt execution for manual review.

1.  When a workflow enters the `AWAITING_APPROVAL` status, navigate to the **Approvals** page.
2.  Review the code, schemas, or security scan reports.
3.  Click **Approve** to proceed or **Reject** to trigger an automatic retry or agent rollback.

---

## 5. Cost Optimization Dashboard

1.  Navigate to the **Cost** page.
2.  View the breakdown of estimated API costs (OpenAI, Gemini, Anthropic token usage), compute runtime billing, and DB storage rates.
3.  Review recommendations generated by the FinOps agent (e.g. downsizing idle clusters, enabling semantic caching).

---

## 6. Downloading Code Artifacts

1.  Once the workflow completes with `SUCCESS`, click the **Download Project** button on the workflow execution page.
2.  The system packages all generated database scripts, schemas, backend codes, nextjs components, and deployments into a single download zip file.
